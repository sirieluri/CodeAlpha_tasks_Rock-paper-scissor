# ✊ Rock Paper Scissors Game

A fun and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. The game features an attractive UI, score tracking, winner announcements, responsive design, and celebratory confetti effects.

## ✨ Features

- 🎮 Play Rock, Paper, Scissors against the computer.
- 🤖 Random computer choice.
- 📊 Live score tracking.
- 🏆 First player to reach **10 points** wins.
- 🎉 Confetti animation when a player wins.
- 🔄 Play Again button to restart the game.
- 📱 Responsive design for Desktop, Tablet, and Mobile.
- 📖 Built-in game rules and instructions.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Canvas Confetti Library

## 🎯 Game Rules

- ✊ Rock beats Scissors.
- ✌️ Scissors beats Paper.
- ✋ Paper beats Rock.
- Same choices result in a tie.
- First player to score **10 points** wins the match.
